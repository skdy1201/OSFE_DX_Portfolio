## 포폴 구성

1.타이틀
2.캐릭터 선택
3.사망
4.엔딩
5.선택 - 다음 레벨로 가거나, 보상을 고르는 것
 
 
 6.전투 및 이벤트

전투 - 일반적인 전투
구조요청 - 3가지 형태의 방이 있고, 오브젝트를 파괴해 인질을 살리면 보상을 받는다.
재해 - 특정 조건이 달린 전투(30초후 폭발 등)
캠프 - 휴식. 힐 및 행운 스텟 증가
상점 - 상점주인에게서 구매, 혹은 전투 가
보물 - 상자방.보상이 있다.
미니보스 - 엘리트방. 일반 전투 보다 강하거나 많은 적이 나온다.
보스 - 해당 월드의 보스가 나온다.
